✦ About This Project

What is DevProfile?

A Django-powered registration system that collects, validates, and stores developer profiles.

Built with Django Forms

This project demonstrates how Django handles form creation, validation, and database storage — core skills every Django developer needs.

Every field in the form is validated on the server side before any data touches the database. Invalid data is rejected with clear error messages shown right next to the field.

Once valid, data is saved to a SQLite database using Django's ORM — no raw SQL needed.

How it works

1

Visit Page

GET request creates an empty form and sends it to the template.

2

Fill Form

User fills in name, email, skills, gender, date of birth.

3

Submit

POST request sends data to the view with a CSRF token.

4

Validate

Django checks every field. Errors are shown if anything is wrong.

5

Save & Redirect

Valid data is saved to the database and user is redirected to success.
